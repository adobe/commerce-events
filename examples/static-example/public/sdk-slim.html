SDK Slim Example

This example does not include the @adobe/adobe-client-data-layer package. Since the acdl is on the window object and the sdk is looking for the object on the window, we can exclude the package in our build and allow for smaller bundle sizes and non-duplication of libraries on the page.

In the console, run "window.commerceEventsSdk.context.getContext();" and you will see our pre-populated events, confirming the sdk is using the acdl properly.
